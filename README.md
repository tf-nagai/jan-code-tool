# 📦 JANコード追加ツール

在庫CSVファイルにJANコードを自動追加するWebツールです。

## 🎯 使い方

### 1. CSVファイルを準備

以下の列を含むCSVファイルを用意してください：

| 必須列名 | 説明 |
|---------|------|
| ブロック略称 | ブロックの略称 |
| ロケーション | ロケーション情報 |
| 商品ID | 商品の識別番号（数値） |
| 在庫数(引当数を含む) | 在庫の数量 |

### 2. ファイルをアップロード

ツールのページでCSVファイルを選択、またはドラッグ&ドロップします。

### 3. 処理開始

「処理開始」ボタンをクリックすると、自動的に処理が開始されます。

### 4. ダウンロード

処理完了後、以下のフォーマットのCSVが自動ダウンロードされます：

| 列番号 | 列名 | 内容 |
|-------|------|------|
| 1 | ブロック | ブロック略称の値 |
| 2 | ロケーション | ロケーション情報 |
| 3 | 商品ID | 商品ID |
| 4 | JAN | 自動生成されたJANコード（13桁） |
| 5 | 在庫数 | 在庫数(引当数を含む)の値 |
| 6 | 出力日付 | 処理実行日（YYYYMMDD形式） |

## 📋 入出力例

### 入力CSVの例

```csv
ブロック略称,ロケーション,商品ID,在庫数(引当数を含む)
A,A-01,3501077,100
B,B-02,44711,50
C,C-03,10000059,200
```

### 出力CSVの例

```csv
ブロック,ロケーション,商品ID,JAN,在庫数,出力日付
A,A-01,3501077,2000035010775,100,20251115
B,B-02,44711,2000000447117,50,20251115
C,C-03,10000059,2000100000595,200,20251115
```

## 🔢 JANコード生成ルール

このツールは**インストアコード（店舗内コード）**を生成します。

### 生成方法

```
商品ID: 3501077
    ↓
8桁に変換: 03501077
    ↓
プレフィックス追加: 200003501077
    ↓
チェックデジット計算: 5
    ↓
最終JANコード: 2000035010775
```

### 特徴

- **プレフィックス**: `2000` (インストアコード)
- **商品ID部分**: 8桁（左側を0で埋める）
- **チェックデジット**: JAN-13標準アルゴリズム
- **総桁数**: 13桁

### 注意事項

⚠️ このツールで生成されるJANコードは**インストアコード**です：

- ✅ 自社の在庫管理システムで使用可能
- ✅ 自社店舗のPOSシステムで使用可能
- ❌ Amazon等の大手ECモールでは使用不可
- ❌ 他社のシステムでは認識されません

全国流通や大手ECモールでの使用には、GS1 Japanへの申請が必要です。
